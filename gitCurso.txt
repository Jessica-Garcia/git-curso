
Arquivo da aula de git e github:

Para subir um repositório que já tenho inicializado e comitado no git local no  para o github


Criar um novo repositório no github e executar os seguintes comandos:

git remote add origin + endereço do repositório utilizando ssh
git push -u origin master (na primeira vez depois somente git push)


Subir as edições do arquivo para o github:

Editar o arquivo

Verificar com git status

Para adicionar direto e comitar 
git commit -am “mensagem”

Para subir para o github
git push origin + branch em que eu estou (neste caso é master)
git push origin master
